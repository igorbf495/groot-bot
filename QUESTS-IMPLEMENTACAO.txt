💡 COMO FUNCIONA
════════════════════════════════════════════════════════════════

1️⃣  INICIAR
    Jogador usa: !quests
    └─ Sistema gera quest do dia (muda a cada 24h)
    └─ Mostra objetivo, dificuldade, recompensa

2️⃣  PROGRESSO
    Jogador joga normalmente (caça, duela, compra, etc)
    └─ Sistema detecta ações relevantes
    └─ Incrementa progresso automaticamente
    └─ Mostra barra de progresso

3️⃣  COMPLETAR
    Quest atinge 100%
    └─ Status muda para "Completada"
    └─ Jogador usa: !completarquest
    └─ Recebe XP + Ouro + ELO (variável)

4️⃣  RESET
    Novo dia (24h depois)
    └─ Nova quest é gerada
    └─ Quest anterior vai para histórico
    └─ Tudo automático!
